## flex class

| 类名     | 说明                            |
| -------- | ------------------------------- |
| `flex`   | 弹性布局                        |
| `flexJB` | justify-content: space-between; |
| `flexJE` | justify-content: space-evenly;  |
| `flexJA` | justify-content: space-around;  |
| `flexJL` | 水平左对齐                      |
| `flexJC` | 水平居中                        |
| `flexJR` | 水平右对齐                      |
| `flexDC` | 垂直居中                        |
| `flexDS` | 垂直顶对其                      |
| `flexDE` | 垂直底对其                      |
| `flexC`  | 水平垂直居中                    |
